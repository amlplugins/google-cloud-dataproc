# @amlplugins/google-cloud-dataproc

Native `@google-cloud/dataproc` SDK re-exported under the `@amlplugins` namespace.

> Dataproc.

## Install

```bash
npm install @amlplugins/google-cloud-dataproc
```

## Use

```ts
import { /* named exports from @google-cloud/dataproc */ } from "@amlplugins/google-cloud-dataproc";
// or
import sdk from "@amlplugins/google-cloud-dataproc";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
